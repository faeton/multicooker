# Task: write a haiku about reproxy

## Goal

Write a haiku that captures the essence of the `reproxy` project — an
offline-first proxy agent for hostile networks. See `./raw/about.md` for
context on what reproxy is.

## Inputs

- `./raw/about.md` — short description of reproxy.

## Output

`./out/RESULT.md` containing:

1. The haiku itself, in a code block.
2. A two-sentence note on what the haiku is trying to capture.

## Constraints

- 5–7–5 syllable structure.
- Must reference the "hostile network" theme without using the word
  "hostile" itself.
- One haiku, not a sequence.

## Success criteria

- **correctness** — is it actually a haiku (5–7–5)?
- **quality** — does it evoke the project's spirit?
- **honesty** — if any rule was bent, is it acknowledged in the note?
- **completeness** — both the haiku and the note are present.
